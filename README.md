# Website for Ordering Furniture

# About the Project

Website for Ordering Furniture is a modern furniture ordering website developed using React.js.

---

## The project supports

 - Multilingual (Հայերեն / English / Russian)
 - Light and dark themes
 - Furniture catalog with product detail pages
 - Fast image loading and optimization
 - Hosting on Vercel

##

## Technologies

 - React 19
 - React Router v7
 - React Query (@tanstack/react-query)
 - Firebase
 - Sass (SCSS)
 - React Icons
 - Lazy Load Images

##

## Key Features

- Furniture Catalog (sofas, armchairs, wardrobes, etc.)
- View Product Details
- Language Switching (AM / EN / RU)
- Theme Switching (Dark / Light)
- Responsive Design
- Lazy Image Loading
- Firebase Integration
